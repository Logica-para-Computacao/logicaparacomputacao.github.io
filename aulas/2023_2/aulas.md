## Exemplos de argumentos simples

- Tema: Sócrates

```
  Sócrates é um homem.
  Todo homem é mortal.
∴ Sócrates é um mortal.
```

- Tema: Sonhos

```
  Meus sonhos são lúcidos.
  Tudo que é lúcido é bom.
∴ Meus sonhos são bons.
```

- Tema: Seres humanos

```
  O ser humano é um ser vivo.
  Todo ser vivo tem células.
∴ O ser humano tem células.
```
